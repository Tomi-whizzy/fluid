⚡ Fluid SDK – WalletConnect Universal Signing

This page loads the Fluid client via a plain <script> tag and the Stellar SDK from a CDN. It demonstrates the standard WalletConnect bindings for universal signing: building the Stellar requiredNamespaces, parsing the connected account, and signing a transaction through the WalletConnectSigner — all behind the same WalletSigner interface that Freighter, the Stellar Wallets Kit, or an in-process keypair also satisfy.

To keep the demo self-contained, a mock WalletConnect provider signs with a throwaway keypair instead of a paired mobile wallet. Swap it for a real @walletconnect/sign-client instance to talk to a live wallet.
