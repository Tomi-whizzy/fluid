# Enterprise Directory Sync (SAML/Okta SSO)

Fluid supports Single Sign-On (SSO) integration using the SAML 2.0 protocol, allowing enterprise identity management suites like Okta, Azure AD, or Ping Identity to authenticate dashboard admins.

## Highlights

1. **Enterprise Directory Sync**:
   - Automatically syncs group membership claims from SAML assertions to Fluid's local admin roles (`SUPER_ADMIN`, `ADMIN`, `READ_ONLY`, `BILLING`).
   - Just-In-Time (JIT) provisioning: Automatically creates administrative accounts upon their first SSO login.
2. **Key Rotation & Multi-secret JWT Support**:
   - Sessions generated by SSO utilize standard cryptographically secure JWT tokens matching existing dashboard requirements.

## Endpoints

- `GET /admin/auth/saml/login`: Redirects browser to Okta for authentication.
- `POST /admin/auth/saml/callback`: Assertion Consumer Service (ACS) endpoint processing SAMLResponses.
